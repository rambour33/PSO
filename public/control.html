? Comment créer une clé API start.gg

1
Se connecter à start.gg

Va sur start.gg ↗ et connecte-toi à ton compte (ou crée-en un gratuitement).

2
Ouvrir les paramètres développeur

Clique sur ton avatar en haut à droite → "Developer Settings".
Accès direct : start.gg/admin/profile/developer ↗

3
Créer un token personnel

Dans la section "Personal Access Tokens", clique "Create token". Donne-lui un nom (ex. PSO Overlay) et clique "Create".

⚠️ Le token n'est affiché qu'une seule fois. Copie-le immédiatement avant de fermer la fenêtre.
4
Coller le token ci-dessous

Colle le token dans le champ ci-dessous et clique "Enregistrer".

Connexion YouTube

Connecté — tokens renouvelés automatiquement

? Comment obtenir ton Client ID et Client Secret Google

1
Créer un projet Google Cloud

Va sur console.cloud.google.com ↗. En haut à gauche, clique sur le sélecteur de projet → "Nouveau projet". Donne-lui un nom (ex : PSO Overlay) et clique "Créer".

2
Activer l'API YouTube Data v3

Dans le menu, va dans APIs et services → Bibliothèque. Recherche "YouTube Data API v3", clique dessus puis "Activer".

💡 Sans cette étape, la connexion échouera avec une erreur "API not enabled".
3
Configurer l'écran de consentement OAuth

Va dans APIs et services → Écran de consentement OAuth. Choisis le type "Externe", clique "Créer". Remplis uniquement :

Nom de l'applicationPSO Overlay
E-mail d'assistanceTon adresse Gmail
Coordonnées du développeurTon adresse Gmail

Clique "Enregistrer et continuer" jusqu'à la fin (les autres étapes peuvent rester vides). Reste en mode Test — pas besoin de soumettre pour validation.

💡 En mode Test, seules les adresses ajoutées dans "Utilisateurs de test" peuvent se connecter. Ajoute ton adresse Gmail.
4
Créer les identifiants OAuth

Va dans APIs et services → Identifiants. Clique "+ Créer des identifiants" → "ID client OAuth". Remplis :

Type d'applicationApplication Web
NomPSO Overlay
URI de redirection autoriséshttp://localhost:3002/auth/youtube/callback

Clique "Créer". Une fenêtre affiche ton Client ID et ton Client Secret — copie-les.

⚠️ Le Client Secret n'est affiché qu'à la création. Si tu le perds, il faudra en créer un nouveau.
5
Coller les identifiants ici et se connecter

Remplis les champs ci-dessous, clique "Enregistrer" puis "Se connecter avec YouTube". Autorise l'accès en lecture seule à ta chaîne.

Panneau de contrôle
Chargement…
Ouvre dans un navigateur sur n'importe quel appareil du réseau
🖥️
PC Serveur
Lance PSO (npm start)
Réseau local (Wi-Fi / Ethernet)
💻
PC Client
OBS / Navigateur
1
Connecter les deux PCs au même réseau
Les deux machines doivent être sur le même réseau local — Wi-Fi domestique ou câble Ethernet. Un hotspot mobile ne fonctionnera pas si les appareils sont isolés.
2
Relever l'adresse IP du PC serveur
L'adresse IP est affichée dans le cadre ci-dessus (ex : 192.168.1.42). C'est l'IP à utiliser depuis le PC client. Si plusieurs adresses apparaissent, essaie la première.
3
Ouvrir l'overlay depuis le PC client
Sur le PC client, ouvre un navigateur ou OBS et entre l'URL suivante :
http://<IP>:3002/overlay
Remplace /overlay par l'overlay voulu (ex : /overlay-slim, /frames, /ticker…).
4
Ajouter dans OBS (source Navigateur)
Dans OBS sur le PC client : Source → Navigateur. Colle l'URL ci-dessus dans le champ URL. Largeur : 1920, Hauteur : 1080.
5
Contrôler depuis n'importe quel PC
Le panneau de contrôle est aussi accessible depuis le PC client :
http://<IP>:3002/control
Dépannage
  • L'overlay ne se charge pas — vérifie que le serveur PSO tourne bien sur le PC serveur (npm start), et que le pare-feu Windows autorise le port 3002 en entrée.
  • Autoriser le pare-feu — Panneau de configuration → Pare-feu Windows Defender → Autoriser une application… → Ajouter Node.js ou ouvrir le port 3002 manuellement.
  • Plusieurs IP affichées — essaie chaque adresse jusqu'à ce que l'overlay réponde. Généralement la 192.168.x.x est la bonne.
  • Connexion intermittente — préfère une connexion Ethernet plutôt que Wi-Fi pour plus de stabilité pendant un stream.
