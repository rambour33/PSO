🎮 Stream Deck — URLs PSO

Cliquer une URL pour la copier · Le tab se ferme seul après l'appui

← Panneau de contrôle
✅ Aucun plugin requis. Utilise l'action Website intégrée dans Stream Deck (catégorie System). Colle l'URL ci-dessous — le navigateur se ferme instantanément après chaque appui.
Comment configurer
1Ouvrir Stream Deck

Lance le logiciel Elgato Stream Deck et sélectionne un bouton vide sur ton deck.

2Action "Website"

Dans le panneau de droite, cherche Website (catégorie System) et glisse-la sur le bouton.

3Coller l'URL

Clique une URL dans la liste ci-dessous pour la copier, puis colle-la dans le champ URL du bouton.

✓Appuyer

L'overlay s'anime avec le type et la durée configurés dans le panneau Animations Entrée/Sortie.

show — affiche l'overlay (même s'il est déjà visible)
hide — masque l'overlay
toggle — bascule visible/masqué
