🚀 Premiers pas

PSO est un système d'overlays pour stream de tournois SSBU. Il tourne sur ton PC en serveur local et communique en temps réel avec OBS via des Browser Sources.

Démarrer le serveur

1
Lancer PSO

Double-clique sur start.bat dans le dossier PSO, ou lance npm start dans un terminal.

2
Ouvrir le panneau

Va sur http://localhost:3002/control dans ton navigateur. Garde cet onglet ouvert pendant tout le stream.

3
Ajouter les overlays dans OBS

Chaque overlay est une URL à ajouter comme Browser Source 1920×1080 dans OBS.

💡Le panneau et OBS communiquent via WebSocket — toute modification dans le panneau s'applique instantanément dans l'overlay, sans rechargement.

Architecture en un coup d'œil

Panneau de contrôle

Interface web sur ton PC — tu y configures tout en live.

Overlays OBS

Browser Sources 1920×1080, fond transparent, en calques dans OBS.

Serveur local

Node.js port 3002 — synchronise panneau et overlays via Socket.IO.

Config persistante

Tout est sauvegardé dans config.json — aucune perte au redémarrage.

🎬 Configuration OBS

Ajouter un overlay

1
Nouvelle source

Dans OBS, clique + dans la liste des sources → Navigateur.

2
Coller l'URL

Copie l'URL depuis l'onglet Liens overlays du panneau et colle-la dans le champ URL.

3
Régler la taille

Largeur : 1920, Hauteur : 1080. Coche Contrôler l'audio via OBS si besoin.

4
CSS personnalisé

Laisse vide. Le fond transparent est géré automatiquement par chaque overlay.

⚠️Ne jamais cocher "Actualiser le navigateur quand la scène devient active" — ça interrompt les animations et l'état en cours.

🏆 Scoreboard

Le scoreboard est le cœur du logiciel. Il affiche les noms, personnages, scores et infos du match en temps réel.

Joueurs & personnages

Tag, nom, pronoms

Tag = le nom court affiché en gros. Nom = nom complet (optionnel). Pronoms = affichés en petit sous le tag.

Personnage

Cherche un personnage dans le champ de recherche. Sélectionne sa couleur de skin avec les pastilles (1 à 8). L'image s'affiche automatiquement dans l'overlay.

Drapeau

Cherche un pays par nom ou code ISO. Le drapeau apparaît dans l'overlay à la position X/Y définie.

Réseaux sociaux

Jusqu'à 3 comptes par joueur (Twitter/X, Twitch, YouTube). S'affichent en carrousel sous le tag.

Couleur joueur

Chaque joueur a une couleur primaire utilisée pour les accents visuels. Peut être synchronisée avec la couleur du stock icon sélectionné.

Infos du match

Événement & Phase

Saisis le nom du tournoi et la phase (Pool A, Top 8, Grands Finals…). Apparaissent en haut du scoreboard.

Format

Choisis entre Bo1, Bo3, Bo5 ou Custom. Affecte le nombre de jeux affichés.

Logo central

Colle une URL d'image ou upload un fichier. Le logo s'affiche au centre entre les deux joueurs. Gère la taille et l'opacité.

Scores

Clique + / − ou saisis directement le score. Reset remet les deux scores à 0. Inverser joueurs échange J1 et J2 avec leurs données.

💡Si tu es connecté à start.gg, un bouton Envoyer sur start.gg apparaît pour reporter le score directement sur la plateforme.

🎙️ Commentateurs

Affiche les noms et réseaux sociaux des commentateurs en overlay.

Configurer les casters

Remplis Nom, Twitter/X, Twitch et YouTube pour chaque caster. Seuls les champs remplis s'affichent.

Disposition

Ligne = côte à côte. Colonne = empilés verticalement. Choisis selon la place disponible dans ta mise en page.

Afficher / Masquer

Utilise le bouton de visibilité ou configure une animation dans l'onglet Animations Entrée/Sortie.

💡Le Créateur de casters permet de concevoir un layout entièrement personnalisé avec formes, textes et calques, si le style par défaut ne te convient pas.

🗺️ Stage Veto

Gère le processus de sélection de stage en direct, visible par les joueurs et les viewers.

1
Configurer le ruleset

Dans l'onglet Stage Veto, choisis la banlist et la séquence de ban (Game 1 et jeux suivants).

2
Lancer le veto

Clique Nouveau veto. Utilise le shifumi si nécessaire pour décider du premier bannisseur.

3
Bannir les stages

Clique sur les stages à bannir dans l'ordre. L'overlay se met à jour en temps réel.

4
Confirmer le stage

Le stage restant est automatiquement mis en évidence. Confirme pour passer à l'overlay "stage sélectionné".

⚠️Active la Stage clause pour empêcher de jouer deux fois sur le même stage dans un même set.

📷 Cam

Overlay pour afficher une ou deux caméras de joueurs avec leurs infos en bas de l'écran.

Taille & position

Règle la largeur et la hauteur avec les sliders ou les presets (320×240, 360×270…). La position X/Y se règle depuis le centre de l'écran. Des presets de position (gauche, centre, droite) facilitent le placement.

Ratio verrouillé

Active le cadenas pour contraindre les proportions (16:9, 4:3, 1:1, 9:16).

Infos joueur

Affiche tag, pronoms, seed et réseaux sociaux sous ou au dessus de la cam. Sélectionne J1, J2 ou les deux.

💡Dans OBS, la cam overlay est transparent — place ta Capture vidéo (webcam) juste en dessous dans la liste des sources, redimensionne-la pour qu'elle corresponde à la zone de la cam overlay.

🖼️ Cadres

Affiche jusqu'à 6 cadres décoratifs pour encadrer tes sources vidéo dans OBS.

Nombre de cadres

Active de 1 à 6 cadres avec les cases à cocher. Chaque cadre a sa propre position (X/Y depuis le centre), taille et ratio.

Presets de disposition

Des boutons de preset arrangent automatiquement les cadres : cam bas-gauche, cam bas-centre, 2 côte à côte, 3 colonnes, grille 4 ou 6.

Label

Chaque cadre peut avoir un label texte (ex: "Player 1"). Règle la position du label (dessus/dessous).

📰 Bandeau défilant

Bande d'informations qui défile en bas (ou en haut) de l'écran.

Messages

Saisis un message par ligne. Ils défilent en boucle séparés par le séparateur choisi (◆, ●, ★, //, · ou personnalisé).

Vitesse

Slider de 20 à 400 px/s. 80 px/s est une vitesse confortable pour la lecture.

Position

Bas = bande en bas de l'écran (par défaut). Haut = en haut. L'animation d'entrée/sortie glisse depuis le bord correspondant.

⏱️ Minuteur

Compte à rebours ou chronomètre affichable en overlay.

Mode compte à rebours

Définis la durée en minutes/secondes. Le minuteur s'arrête automatiquement à 0:00 et change de style à l'approche du temps imparti (zone d'alerte configurable).

Mode chronomètre

Démarre depuis 0, monte indéfiniment. Utile pour chronométrer des manches ou des temps de réaction.

Contrôles

Start / Pause / Reset. Le minuteur continue même si tu changes d'onglet dans le panneau.

Style

Trois styles : Défaut (boîte avec fond), Minimal (chiffres seuls), Grand (très grand affichage).

⚔️ VS Screen

Écran de présentation affiché entre les sets pour annoncer les joueurs.

Image de fond

Upload une image (PNG/JPG/WebP) ou colle une URL. Applique des filtres (luminosité, contraste, saturation) directement depuis le panneau.

Particules thème

Les particules du thème actif s'affichent automatiquement en surcouche. Désactivable.

Durée d'affichage

Configure une durée en secondes. L'overlay se masque automatiquement après ce délai.

🎛️ Studio — Créateur de scènes

Le Studio permet de composer tous tes overlays en une seule Browser Source. Idéal pour OBS avec des scènes complexes.

Gérer les scènes

Tu peux créer jusqu'à 9 scènes indépendantes. Chaque scène a sa propre URL : /super-overlay/1 à /super-overlay/9. L'URL /super-overlay (sans numéro) suit la scène active dans le Studio.

Rename chaque scène en double-cliquant sur son onglet. Les scènes sont sauvegardées et persistent au redémarrage du serveur.

✅Dans OBS, ajoute 9 Browser Sources (/super-overlay/1 à /super-overlay/9) dans des scènes OBS différentes. Chacune est indépendante et peut être active simultanément.

Calques & positions

Activer un calque

Coche/décoche les overlays dans la liste. L'ordre de la liste définit le z-index (premier = en dessous, dernier = au dessus).

Déplacer un overlay

Dans le canvas de prévisualisation, glisse le badge coloré d'un overlay pour le repositionner. Les coordonnées X/Y sont modifiables aussi par les champs numériques.

Opacité par calque

Chaque calque a son propre slider d'opacité (0–1). Utile pour des overlays en transparence sur d'autres.

Snapshot

Le bouton 📷 sur un calque sauvegarde l'état visuel actuel de cet overlay. La miniature s'affiche dans le canvas même si l'overlay est dynamique.

Fond & particules

Couleur de fond

Choisis parmi les presets (transparent, noir, bleu nuit), la couleur du thème actif, ou une couleur personnalisée via le color picker.

Image / Texture

Upload un fichier image ou une texture. Mode Image = plein écran centré. Mode Texture = répété en mosaïque. Règle le mode de fusion (normal, multiply, overlay, screen…) et l'opacité.

Particules

Active les particules du thème actif. Contrôle l'opacité et la quantité avec les sliders.

🎨 Thèmes visuels

Les thèmes changent les couleurs de tous les overlays simultanément en un clic. Plus de 20 thèmes prédéfinis + créateur custom.

Appliquer un thème

Clique sur une carte de thème dans l'onglet Thèmes. Le changement est instantané sur tous les overlays actifs.

Default

Or chaud sur fond sombre. Le thème de base.

Cyberpunk

Cyan néon sur fond très sombre.

Synthwave

Rose/violet rétro années 80.

Midnight

Bleu nuit profond.

Fire / Water / Eco

Thèmes élémentaires (rouge, bleu clair, vert).

Transparent

Fond semi-transparent, pour composition avancée.

Créer un thème personnalisé

Dans Créateur de thème, configure :

• Type de fond : couleur unie, dégradé ou texture
• Couleur accentuation : utilisée pour les titres et séparateurs
• Couleurs J1/J2 : couleurs par défaut des joueurs
• Couleurs texte : nom, tag, pronoms

Clique Sauvegarder pour l'ajouter à ta liste de thèmes. Il sera disponible dans toutes les sessions suivantes.

💡Le thème Dual permet d'utiliser une couleur différente pour chaque joueur, synchronisée avec leur couleur de personnage.

🎮 Start.gg

PSO se connecte à start.gg pour récupérer automatiquement les participants, sets et résultats.

Connexion

1
Obtenir une clé API

Va sur start.gg → Développeur → Clés API et crée une clé personnelle.

2
Coller la clé

Dans l'onglet Start.gg Overlays, colle ta clé API et clique Sauvegarder.

3
Chercher un tournoi

Saisis le slug du tournoi (ex: tournament/smash-odyssey-5) et clique Rechercher.

4
Charger les participants

Sélectionne l'événement dans le menu déroulant et clique Charger. Les entrants sont importés automatiquement.

Bracket & sets en cours

Sets en cours

La liste des sets actifs/en attente apparaît automatiquement. Clique Charger le set pour remplir le scoreboard avec les deux joueurs.

Envoyer un score

Après un set, clique Envoyer sur start.gg dans le Scoreboard pour reporter le résultat directement sur la plateforme.

Visualiseur Bracket

Affiche l'arbre complet du bracket en overlay. Se met à jour automatiquement quand des résultats sont rentrés.

Head-to-Head

Sélectionne deux joueurs dans les menus déroulants et clique Calculer H2H. PSO récupère l'historique complet des sets entre ces deux joueurs sur start.gg et affiche :

• Bilan victoires/défaites
• Pourcentage de victoire
• Personnages les plus joués par chacun

Clique Afficher pour envoyer le résultat vers l'overlay H2H dans OBS.

💜 Twitch

Intégration complète : viewers, alertes, chat, prédictions.

Connexion

1
Créer une app Twitch

Sur dev.twitch.tv → Console, crée une application. Redirect URI : http://localhost:3002/auth/twitch/callback.

2
Copier Client ID & Secret

Colle-les dans l'onglet Twitch du panneau et clique Sauvegarder.

3
Authentifier le broadcaster

Clique Connecter le compte Twitch pour obtenir les permissions broadcaster (abonnés, prédictions).

Viewers

Compteur en temps réel dans l'overlay /twitch-viewer.

Chat

Overlay de chat live sur /twitch-chat. Filtrable par type de badge.

Alertes

Follows, subs, raids, bits — animés sur /twitch-alerts.

Prédictions

Lance et gère des prédictions Twitch depuis le panneau.

▶️ YouTube

Chat, viewers et alertes pour les streams YouTube Live.

La connexion YouTube utilise OAuth 2. Configure Client ID et Client Secret depuis la Google Cloud Console, puis autorise le compte.

Overlays disponibles : /youtube-chat, /youtube-viewer, /youtube-alerts et /combined-chat (fusion Twitch + YouTube).

💡Le Chat combiné affiche Twitch et YouTube dans un seul overlay avec des badges de couleur différente pour identifier la source.

🎮 Stream Deck

Contrôle l'affichage de tous les overlays directement depuis ton Stream Deck, avec les animations configurées.

Configuration (aucun plugin requis)

1
Ouvrir Stream Deck

Lance le logiciel Elgato Stream Deck et sélectionne un bouton vide.

2
Action "Website"

Dans le panneau de droite, cherche Website (catégorie System) et glisse-la sur le bouton.

3
Coller l'URL

Copie une URL depuis /deck et colle-la. Le navigateur qui s'ouvre se ferme instantanément.

Format des URLs

ActionURLEffet
show/api/deck/cam/showAffiche l'overlay avec l'animation d'entrée
hide/api/deck/cam/hideMasque avec l'animation de sortie
toggle/api/deck/cam/toggleBascule visible/masqué

Overlays disponibles : scoreboard · scoreboard-slim · casters · ticker · cam · frames · timer · stream-title · h2h · player-stats · bracket · top8 · et plus encore — voir la page complète.

✅Les animations respectent les paramètres configurés dans l'onglet Animations Entrée/Sortie. Change le type ou la durée dans le panneau et tous tes boutons Stream Deck utilisent automatiquement la nouvelle config.

🌐 Multi-PC

Utilise PSO sur un PC dédié aux overlays, avec OBS sur un autre PC du même réseau.

1
Connecter les PCs

Les deux PCs doivent être sur le même réseau local (idéalement câble Ethernet pour la stabilité).

2
Relever l'IP du serveur PSO

Dans l'onglet Multi-PC du panneau, les IPs locales disponibles sont listées automatiquement.

3
Ajouter dans OBS (PC client)

Sur le PC OBS, ajoute les Browser Sources avec l'URL http://<IP>:3002/overlay au lieu de localhost.

4
Ouvrir le panneau depuis l'autre PC

Accède à http://<IP>:3002/control depuis n'importe quel PC du réseau.

⚠️Si la connexion échoue, vérifie que le pare-feu Windows autorise le port 3002 sur le PC serveur. Panneau de configuration → Pare-feu → Règles de trafic entrant → Nouveau → Port 3002 TCP.

⚡ Astuces & conseils

Workflow de tournoi optimal

Avant le stream : configure les thèmes, les personnages, connecte start.gg et Twitch. Prépare tes scènes OBS avec tous les overlays en calques.

Pendant les sets : charge le set depuis start.gg en un clic → le scoreboard se remplit automatiquement → lance le Stage Veto → incremente les scores avec +/- → envoie sur start.gg en fin de set.

Entre les sets : affiche le H2H ou les stats joueur pendant les transitions. Utilise le Stream Deck pour afficher/masquer les overlays rapidement.

Conseils de performance

💡Mets le Super Overlay en une seule Browser Source au lieu d'empiler 10 sources dans OBS — ça réduit la charge GPU et le risque de décalage entre les calques.
💡Le Ticker utilise un rendu RAF optimisé. Ne désactive jamais OBS "Actualiser au changement de scène" sur cette source.
✅Toutes les données sont sauvegardées en temps réel dans config.json. Tu peux redémarrer le serveur en cours de stream — les overlays récupèrent leur état en quelques secondes.
⚠️Les images uploadées (fonds, textures, personnages) sont stockées dans public/uploads/. Sauvegarde ce dossier avec config.json pour exporter ton installation complète.
